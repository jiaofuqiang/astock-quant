## 二、六层穿透维度

| 层级 | 维度 | 传感器 | 数据源 | 信号强度 |
|:----|:----|:------|:------|:--------:|
| 🌐 全球层 | 美股/原油/贵金属 | (已在穿透评分的bundle中) | bundle.contradiction_report | — |
| 📈 **大盘层** | **大盘环境** | EnvSensor | contradiction_report | 权重1.0 |
| 💰 **资金层** | **资金流向** | FundFlowSensor | lhb_scoring_cache + lhb_cache.db | 权重1.0 |
| 📡 板块层 | 板块强度/主线 | (已在穿透评分中) | bundle | — |
| 🔍 **个股层** | **K线形态+连续上榜** | BoardPatternSensor + ContinuitySensor | kline_cache.db + lhb_cache.db | 权重0.6+0.6 |
| 🗓️ **时序层** | **季节效应** | SeasonalitySensor | datetime | 权重0.7 |

## 三、维度对穿透矩阵

不是每个维度独立评分求和，而是两两组合成**联合判断**：

| 维度对 | 联合输出 | 置信度 |
|:------|:--------|:------:|
| **资金流向 × 大盘环境** | 🚀强烈做多 / ⚠️谨慎 / 🚨卖出 | 0.3~0.9 |
| **资金流向 × K线形态** | ✅最佳买入 / ❌陷阱 / ⚠️假突破 | 0.35~0.85 |
| **大盘环境 × 季节效应** | 💰重仓 / 💵轻仓 / 🛑空仓 | 0.4~0.9 |
| **K线形态 × 连续上榜** | ⭐龙头首板 / ❌连续风险 | 0.4~0.75 |
| **大盘环境 × K线形态** | 🏆趋势跟踪 / 🔄超跌反弹模式 | 0.4~0.8 |

## 四、传感器核心逻辑

### FundFlowSensor — 资金流向（回测数据注入）

```
信号计算（基于今日回测发现的收益权重）:
  首次上榜+北向买+总净买1~3万    → +50分 (回测+2.40%/64.5%)  ← 最强
  仅北向买(含details)            → +25分 (+1.26%/57.0%)
  北向+机构双买                  → +25+10分 (+0.87%/54.5%)
  机构+量化合力                  → +18分 (+1.62%/72.9%)
  游资+量化合力(无机构)          → -20分 (-0.65%/45.8%)  ← 要回避
  拉萨散户买                     → -15分 (-1.65%/37.6%)  ← 反向指标
```

### BoardPatternSensor — K线形态

```
超跌反弹(前5日跌>10%) → -30分 (坑)
普通上涨/横盘         → +15~20分 (最佳)
高位突破(前5日涨>15%) → +10分
```

### SeasonalitySensor — 月份效应

```
月份周期    → 信号值
季中月(2/5/8/11) → +50~85 (最强)
季初月(1/4/7/10) → +10~15 (中性)  
季末月(3/6/9/12) → -10~-20 (坑！)
```

## 五、实践论闭环链路

```
15:35 (T日收盘)
  └→ maodun_verifier.py 记录红黑榜
      └→ grade_backtest表（T+1收益率写入）
 
18:00 (T日盘后)
  └→ auto_calibration.py 校准阈值
      └→ 读红黑榜 → 算偏差 → 调阈值 → 输出cal_adjustments.json

08:30 (T+1日开盘前)
  └→ morning_pipeline.py (含AI引擎)
      1. AIContextBuilder构建上下文
      2. AIDecisionEngine.decide() ← AI穿透决策
      3. 注入bundle.ai_decision
      4. 打印决策摘要
```

## 七、与morning_pipeline的衔接

morning_pipeline不需要大幅改动，只需在输出时读取 `bundle['ai_decision']`中的AI决策结果，与现有的矛盾评级并行展示，或者**当AI决策与矛盾评级冲突时，以AI决策为准**（因为AI引擎融合了今日发现的最强信号）。
