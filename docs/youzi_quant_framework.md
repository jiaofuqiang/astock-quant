# A股顶级游资量化策略框架

## 一、核心差距分析 — 我们和顶级游资差在哪

### 我们现有的策略
- 看个股K线特征（涨幅/横盘/底部/放量）
- 看板块联动数量
- 评分≥40就买入
- **问题：没有情绪周期、没有卖出纪律、没有仓位管理**

### 顶级游资的做法
- **选股：** 只做龙头，不做跟风。二板定龙头，三板以上确认。
- **买入：** 打板（封板瞬间买入），而不是开盘追入。
- **卖出：** 低于预期竞价就卖、炸板卖、逻辑破坏无条件卖。
- **仓位：** 确定性高时重仓，不确定性时空仓。
- **情绪周期：** 冰点期轻仓/空仓，发酵/高潮期重仓。

## 二、可直接量化的改进点

### 1. 龙头评分（我们当前缺失的）
- 连板高度：2板>1板，3板>2板
- 板块内排名：龙头>跟风
- 涨停时间：早盘涨停>午后涨停
- 封板质量：封单大>封单小
- 龙虎榜席位：知名游资买入>普通营业部

### 2. 情绪周期择时（我们当前缺失的）
- 衰退期（炸板率>40%+连板高度<3板）→ 空仓/极轻仓
- 发酵期/高潮期 → 重仓做多
- 冰点期 → 轻仓试错

### 3. 卖出纪律（我们当前缺失的）
- 低于预期→竞价卖
- 炸板不回封→卖
- 板块龙头断板→跟风全卖
- 逻辑破坏→无条件卖

### 4. 打板买入（我们当前用开盘追入）
- 打板比开盘追入的确定性更高（封板确认后才进）
- 但打板也有问题：炸板风险、买不进去（一字板）

## 三、对现有游资v4模型的改进方案

### 改进1：加龙头地位权重
```
v4评分 × 0.7 + 龙头评分 × 0.3
```
每天扫描时识别：
- 该票在板块内的涨幅排名
- 该票的连板高度（从K线判断连续涨停）
- 该票的涨停时间（盘中用腾讯行情）

### 改进2：加情绪周期过滤器
```
if 情绪周期 == '衰退期':
    仓位 = 0  # 空仓
elif 情绪周期 == '冰点期':
    仓位 = 20%
elif 情绪周期 == '启动期':
    仓位 = 40%
elif 情绪周期 == '发酵期':
    仓位 = 70%
elif 情绪周期 == '高潮期':
    仓位 = 60%
```

### 改进3：加卖出纪律
```
买入后T+1:
  if 低开>3% or 竞价弱: 竞价卖
  if 炸板不回封: 炸板瞬间卖
  if 板块龙头断板: 立刻卖
  else: 持有到10:00看强度
```

## 四、打板策略 vs 现有策略的收益差异

### 我们现在的买点：开盘追入
收益：T+1收+1.82%~+2.13%，但开盘买入后可能被砸

### 顶级游资的买点：打板（封板确认后）
- 优点：确定性高，封板代表当天做多力量确认
- 缺点：炸板就亏、一字板买不进
- 实际收益数据（来自我们自己的回测）：
  - 开盘追入(涨≥7%) → T+1收+1.82%/胜62.0%
  - 如果改为打板买入（排除一字板），收益应该接近涨停组的+1.78%/胜58.2%

### 结论
打板并不会比开盘追入收益更高（我们自己的数据证明了），
但情绪周期择时+卖出纪律+龙头聚焦这三点叠加能大幅提升收益。
