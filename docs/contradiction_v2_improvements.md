# 矛盾引擎v2.0 — 基于8维历史回测的改进清单

## 回测数据基础
- 15988个涨停样本 × 570个交易日 × 3282只主板股票
- kline_cache.db: 182万行K线 + lhb_cache.db: 14493条龙虎榜
- 回测脚本: contradiction_bt_dim1_4.py, contradiction_bt_dim5_8.py
- 完整报告: astock/data/contradiction_bt_full_report.json

## 6项核心改进

### 1. 评级五维化（P0）
- 新增「环境力」因子，作为第5维评分维度
- 高潮(70+)→15分, 发酵(55-69)→12分, 震荡(45-54)→8分, 弱势(35-44)→4分, 冰点(<35)→1分
- 基于维度3回测：外因(大盘)对T+1收益影响远超内因(个股质量)
- 甲等阈值从52→55分（因5维整体提分）

### 2. 量比评分翻盘（P0）
- **改前**：缩量<0.5最佳，放量>1.0预警崩盘
- **改后**：0.7-1.0正常量最佳(+2.88%/65.8%)，缩量<0.5开盘溢价高但收盘回落
- 暴量>1.5仍有+1.76%/55.6%正收益，改为"仍可买入但仓位减半"
- 修改位置：morning_pipeline.py内identify_main_contradiction的个股层量比评分

### 3. 竞价低开>3%仓位减半（P1）
- 维度4回测：低开>3%(矛盾转化)→T+1 -2.80%/33.6%
- 实现：分仓前检查bidding_weak标记，有则cap_total*=0.5
- 仅在交易时段(9-15时)触发

### 4. 冰点期保留8%仓位（P1）
- **改前**：环境<25时空仓0%
- **改后**：保留8%做最强涨停板
- 维度1回测：冰点涨停仍有+0.66%/50.6%正收益

### 5. 方向一致偏多加仓10%（P2）
- 维度2回测：一致偏多→T+1 +1.16%/56.1%
- 实现：当70%+方向为偏多时，cap_total*=1.1（上限85%）

### 6. 环境历史数据回填（基础设施）
- 新脚本: env_daily_backfill.py → 从kline_cache.db构建570天环境数据
- 输出: env_daily_history.json (环境分/涨跌比/涨停强度/跌停数)
- 注入到bundle['env_daily_history']供穿透评分使用
- 解决market_daily.db只有3天的数据荒漠问题

## 修正的旧假设
| 旧信念 | 回测结论 | 新行动 |
|--------|---------|--------|
| 缩量=共识一致最佳 | 正常量0.7-1.0最佳(+2.88%/65.8%) | 关注量能健康的涨停 |
| 高连板=高风险 | 3板最佳(+3.42%/61.2%), 4板+仍有+2.45% | 3板可追,不必因高位清仓 |
| 冰点=空仓 | 冰点涨停+0.66%/50.6%正收益 | 保留8%仓位做最强 |
| 内因>外因 | 外因强+内因弱=+1.82%/64.3% >> 外因弱+内因强=+0.40%/51.5% | 环境分权重提最高 |
| 暴量=崩盘 | 暴量>1.5仍有+1.76%/55.6% | 仍可做,但仓位减半 |
