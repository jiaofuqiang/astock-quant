# 🔥 顶级游资手法 → 可落地的改进方案

基于对炒股养家、赵老哥、章盟主、作手新一、涅槃重生等顶级游资手法的研究，
结合我们现有的游资v4评分模型（全量3,195只主板验证），梳理出可落地的改进点。

## 一、当前策略收益（基准）

| 策略 | T+1收 | 胜率 | 日均信号 |
|:----|:----:|:---:|:-------:|
| 游资v4≥40+涨停+底部 | +2.13% | 63.1% | 6.4次 |
| 游资v4≥40+涨幅≥7%+底部 | +1.82% | 62.0% | 10.5次 |
| 游资v4≥40+板块联动≥10只 | +1.36% | 55.8% | 37.7次 |

## 二、顶级游资 vs 我们的差距

### 差距1：没有情绪周期择时
- 游资：冰点期空仓/轻仓（0-20%），发酵/高潮期重仓（60-80%）
- 我们：无论什么行情都按同样条件扫描买入
- **预期提升：胜率+3-5%，最大回撤-10%**

### 差距2：没有只做龙头
- 游资：只做板块龙头，不做跟风
- 我们：联动≥10只板块的所有票都纳入
- **预期提升：T+1收+0.5%~+1.0%**

### 差距3：没有卖出纪律
- 游资：低于预期竞价卖、炸板卖、逻辑破坏无条件卖
- 我们：只说了"T+1冲高卖"，没有具体规则
- **预期提升：避免亏损扩大，胜率+2-3%**

### 差距4：没有仓位管理
- 游资：确定性高重仓，不确定轻仓，连续亏强制空仓
- 我们：没有仓位概念，等额分配
- **预期提升：牛市多吃肉，熊市少挨打**

## 三、马上能落地的改进（按优先级排序）

### P0：情绪周期过滤器
每天开盘前判断市场状态：
- 发酵期/高潮期 → 正常扫描+买入
- 震荡期 → 只做联动≥10只的板块，仓位减半
- 冰点期 → 空仓，不扫描

判断指标：涨停数、炸板率、最高连板、昨日涨停溢价

### P1：龙头聚焦（板块内只做前3名）
板块联动≥10只时，只做涨幅排名前3的票：
- 龙头（第1名）：重点关注
- 次龙（第2名）：可关注
- 老三（第3名）：谨慎关注
- 跟风（第4名+）：不操作

### P2：加分因子
- 早盘涨停（9:30-10:00涨停的票加分）
- 龙虎榜多路游资同时买入（多席位协同加分）
- 昨日涨停且今日继续大涨（连板股加分）

### P3：卖出规则量化
- 低开>3%→竞价卖
- 炸板不回封→卖
- 板块龙头断板→跟风全卖
- 高开>5%但10分钟内不涨停→卖一半

## 四、预期收益

全部落地后预期:
- 涨停+底部策略：T+1收从+2.13%→+3.0~4.0%（情绪过滤器+龙头聚焦+卖出纪律）
- 板块联动策略：T+1收从+1.36%→+2.0~2.5%（只做龙头+情绪择时）
- 胜率：从63.1%→68-72%
- 最大回撤：从未知→<15%

## 五、不做的理由（理论很美好，实际差距不大）

1. **打板策略**：我们回测显示开盘追入和打板收益接近（+1.82% vs +1.78%），打板还多了一字板买不到的问题
2. **半路策略**：没有日内分时数据，无法精确实现
3. **仓位管理**：用户资金量小（1万），分仓意义不大，但情绪择时仍有意义
