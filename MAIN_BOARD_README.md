## 🔧 全量扫描用法

```bash
# 涨幅排行（5日/20日/60日TOP）
python3 scan_main_board.py --top        # ~5秒

# 成交量异动
python3 scan_main_board.py --volume     # ~5秒

# 金叉信号+多头排列
python3 scan_main_board.py --signals    # ~5秒

# 全量5项分析（涨幅+金叉+多头+RSI超买超卖）
python3 scan_main_board.py --all        # ~5秒

# 默认（涨幅+量异动+金叉+多头）
python3 scan_main_board.py              # ~17秒
```
