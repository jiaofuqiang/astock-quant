# A股多层多维分析预测系统

## 核心思想

**股价 = f(全球宏观, 国内政策, 板块题材, 公司基本面, 个股技术)**

外部因素的**影响逐层传导**，每一层都有多个维度去评估。

```
🌍 全球宏观层
    ↓ 传导
🇨🇳 国内宏观层
    ↓ 传导
🏭 板块题材层
    ↓ 传导
💼 公司行业层
    ↓ 传导
📄 个股技术层
```

## 5个层级

| 层级 | 关注 | 示例 |
|------|------|------|
| 全球宏观 | 地缘政治、美元/美债/黄金/石油、全球股市联动 | 中东冲突→油价 |
| 国内宏观 | 货币政策、财政政策、产业政策、监管、经济数据 | LPR降息→利好地产 |
| 板块题材 | 题材消息面、资金集中度、板块指数 | AI题材、低空经济 |
| 公司行业 | 基本面、订单、技术路线、竞争格局 | 光模块800G→1.6T |
| 个股技术 | 价格位置、趋势、量价关系、筹码分布 | 突破/回踩/超买 |

## 5个维度（每层都做这5个维度的评估）

| 维度 | 含义 | 信号来源 |
|------|------|----------|
| 消息面 📰 | 发生了什么，影响力如何，第几次出现 | 新闻、公告、传闻 |
| 资金面 💰 | 谁在买/卖，量级，方向 | 北向/主力/游资龙虎榜 |
| 情绪面 📊 | 市场心态：恐慌/贪婪/麻木 | 涨跌比、换手率、文章态度 |
| 技术面 📈 | 价格位置、趋势方向、量价配合 | K线、均线、MACD、成交量 |
| 基本面 🏢 | 业绩、估值、成长性、护城河 | 财报、增长率、PE/PB |

## 传导路径识别

核心算法：当外部层出现冲击信号时，自动判断：
1. 冲击强度（1-5）
2. 传导路径（哪些行业/板块会受益/受损）
3. 时间尺度（短期冲击vs长期影响）
4. 当前在预期中的程度（已price in？）
5. 最终影响（利好/利空，买入/卖出/观望）

## 使用方式

```bash
# 完整运行所有层级
python3 run.py

# 只运行特定层级
python3 run.py --layers macro,domestic,sector

# 只看指定个股的多层关联
python3 run.py --stock 300308
```
